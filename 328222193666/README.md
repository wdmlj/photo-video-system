# 小妹的私密故事 - 照片视频管理系统

一个基于React + TypeScript + Tailwind CSS开发的照片视频管理系统，包含前端H5网页端和后台管理端。系统采用粉色系设计风格，提供了照片和视频的分类展示、在线观看、搜索筛选等功能，以及完整的后台管理功能。

## 系统功能

### 前端H5网页端
- 粉色系主视觉设计，低饱和柔粉为主，浅白/淡灰为辅
- 密码验证登录，保障隐私安全
- 顶部横向滚动广告栏（支持自定义）
- 照片按相册分类展示，支持点击放大、左右滑动切换
- 视频支持在线播放、暂停、进度条拖动
- 支持搜索/筛选照片/视频（按名称/分类）
- 响应式设计，适配手机/平板/电脑全终端

### 后台管理端
- 独立管理员登录入口
- 批量上传照片/视频功能，支持自定义分类/命名
- 数据实时查看：总照片数/总视频数/各分类数量/今日访问量/总访问量/最近登录用户
- 数据可视化：柱状图/饼图展示各项数据
- 完整的媒体管理功能：分类编辑/重命名/删除
- 广告栏内容自定义编辑
- 前端登录密码修改，访问记录查看
- 多管理员账号管理，支持权限分配

## 技术栈
- React 18+
- TypeScript
- Tailwind CSS
- React Router
- Recharts (数据可视化)
- Sonner (通知提示)
- 本地存储 (localStorage)

## 部署指南

### 在GitHub Pages上部署
1. 在GitHub上创建一个新的仓库
2. 手动上传项目文件到仓库中
3. 在仓库的Settings -> Pages中设置部署分支为main，路径为根目录
4. 等待GitHub Pages自动部署完成

### 本地开发
1. 安装依赖：`npm install` 或 `pnpm install`
2. 启动开发服务器：`npm run dev` 或 `pnpm dev`
3. 构建生产版本：`npm run build` 或 `pnpm build`
